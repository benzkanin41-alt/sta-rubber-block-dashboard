# STA Rubber Block Export Dashboard

Online static dashboard for Thai rubber block exports, HS `400122`, world scope, sourced from Thailand Ministry of Commerce Trade Report.

## Online

GitHub Pages serves this site from the `main` branch root.

## Files

- `index.html`, `app.js`, `styles.css`: dashboard UI.
- `data/export_data.json`: reviewed dashboard data extract.

## Local Source

The source dashboard and refresh script live in the Codex workspace under `sta_rubber_dashboard`.
